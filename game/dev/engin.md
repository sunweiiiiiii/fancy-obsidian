# 游戏引擎笔记

> 最后更新：2026-04-19

---

## Godot 是什么

Godot 是一个**开源、免费**的游戏引擎，支持 2D 和 3D 游戏开发，可以将游戏导出到 Windows、Linux、macOS、手机、Web 等多个平台。

---

## Godot 与主流游戏引擎对比

| | **Godot** | **Unity** | **Unreal Engine** |
|---|---|---|---|
| 费用 | 完全免费开源 | 免费但有收入抽成 | 免费但有收入抽成 |
| 体积 | 极小（~100MB） | 很大（几 GB） | 非常大（10GB+） |
| 脚本语言 | GDScript（类 Python）/ C# | C# | C++ / Blueprints |
| 2D 支持 | ⭐⭐⭐⭐⭐ 原生专为 2D 设计 | ⭐⭐⭐ 2D 是附加功能 | ⭐⭐ 偏 3D |
| 3D 支持 | ⭐⭐⭐ 中等 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 上手难度 | 简单 | 中等 | 较难 |
| 适合场景 | 独立游戏、2D 游戏 | 中小型全类型 | 3A 大作 |

---

## 为什么本项目选用 Godot

当前项目（坦克大战双人合作游戏）选用 Godot 的理由：

- **原生 2D 支持**：Godot 的 2D 引擎效率高，坐标系和物理都针对 2D 专门优化
- **GDScript 语法简洁**：类似 Python，修改和迭代速度快
- **轻量**：安装包仅约 100MB，启动快，不占用大量资源
- **完全免费**：无收入抽成，适合独立开发
