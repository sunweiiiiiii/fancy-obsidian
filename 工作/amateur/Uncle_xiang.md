## 毛毯清洗设备 · 机械图纸设计文档 (V3.1 干湿分离全量版)

---

### 一、设备概述

本设备用于清洗铺放在基座台面上的重型工业毛毯。活动轴（龙门架）携带四组独立作业阵列，沿长轴方向往复运动完成“干吸、喷洗、刷洗、强排脱水”全流程作业。
本设备在原机械结构基础上，升级集成了**顶部影像扫描、左侧独立动力岛、全封闭工业拖链、矩阵分段控制以及自动化卷收系统**。为彻底规避管路淤塞，首创**“干湿分离”独立吸尘循环**。污水经台面四周具有微型起坡的导流槽收集后排出，实现高能效、高脱水率的本质安全型无人化清洗。

### 三、核心结构详述

#### 3.1 基座与自动卷收单元
- **立柱与台面**：四角各一根（80×80mm），三层横梁加固。绝对平整的不锈钢台面中心进行 5-10mm 的微型“龟背”起坡处理，利用重力加速污水向四周回流，避免中心积水。
- **不锈钢围板 & 导流槽**：四周完全密闭无透空间隙。导流槽沿前后左右四边布置（宽 160mm，高 70mm），将废水向四角汇集。
- **角落漏斗导水口 × 4**：四角各一个正四棱锥漏斗，旋转 45° 对角放置。漏斗口上方加装**提篮式粗滤网**，拦截泥沙与动物毛发。
- **减阻机构（隐藏式顶升滚筒）**：台面下预埋多根长向钢制滚筒。清洗时滚筒低于台面，保持台面绝对刚性（满足刷洗和真空吸水的密闭需求）；卷收时由气缸将滚筒顶出 30mm 架空毛毯，变滑动摩擦为滚动摩擦。
- **卸料端卷收机**：基座 X 轴末端外侧安装全宽动力卷筒。配合龙门架气动机械夹爪递送毛毯，自动卷取卸料。

#### 3.2 工作台框架与随动系统
- **顶部框架**：四根工作台立柱（高 500mm）立于基座四角顶面。仅保留两侧纵向导轨（沿 X 方向），不设横向端梁，确保龙门架完整走完全程。
- **全封闭静音工业拖链**：在导轨外侧平行增加 U 型槽。龙门架的高压水管、气管、主控线、大口径抽风管全部包裹于拖链内，随龙门架做恒定弯曲半径的滚动折叠，彻底杜绝管线悬空与缠绕。

#### 3.3 动力与环境布局（左侧动力岛）
为符合工业 LOTO（锁牌停电）安全标准，消除涉水区触电隐患及震动干扰，实行本质安全设计：
- **电气控制柜**：IP55 防护等级，内置 PLC、变频器。面板配拍击式急停开关及 LOTO 挂锁点。
- **干吸单元动力**：高压风机 + 倒锥形旋风分离集尘桶。集中且干爽地处理第 1 排吸入的毛发灰尘。
- **湿抽单元动力**：水环式真空泵 + 气液分离罐。处理第 4 排抽取的污水。**干湿管路绝对物理隔离**。
- **安全走线**：动力岛至机床的主管线经由地下带盖板的地沟接入机床，消除车间绊倒风险。

#### 3.4 顶部影像与 AI 控制系统
- **硬件布置**：独立顶部桁架中央安装千万像素级工业面阵相机（Z轴高度电动可调适配视场），四周布置 LED 无影漫反射面板灯，消除湿润毯面的点光源反光。
- **AI 联动**：扫描生成毛毯污渍热力图坐标，下发给 PLC 执行“矩阵式局部清洗”，大幅降本增效。

#### 3.5 四组作业阵列（干湿解耦的矩阵控制）
四组阵列沿长轴（X 方向）前后并排，间距 180mm。本方案实现了**三套独立的升降逻辑 (Z轴 A/B/C)**：

| 顺序 | 阵列类型 | 升降控制 | 结构与矩阵控制逻辑 |
|---|---|---|---|
| **第 1 排** | 干式吸尘单元 | **独立 Z轴 A** | 20个**鸭嘴式扁平宽口吸头**。各吸口带气动翻板阀，受 AI 启停。上方设**变径主歧管**汇集风力，中心接大波纹管走拖链至外置集尘桶。干吸时不与水路接触。 |
| **第 2 排** | 喷水阵列 | **联动 Z轴 B** | 20个宽扇面喷头，锥形水帘覆盖微交叉。每 2-3 个喷头共用电磁水阀，由 AI 调度仅在污渍正上方局部出水。 |
| **第 3 排** | 羊毛刷阵列 | **联动 Z轴 B** | 30个圆盘刷（12 根外刷丝+绒毛环）。分为 5-6 个独立动力模组，各模组配旋转电机与气动下压装置，刷洗已湿润区域。 |
| **第 4 排** | 复合脱水头 | **独立 Z轴 C** | 1组横跨全宽。前置**真空强排长缝**（接气液分离罐）；后置紧跟 **45°倾斜高压风刀**（迎风面剥离水膜，向推射）。脱水率 >80%。 |

### 四、八阶段工作动画循环（干湿分离逻辑）

为杜绝吸口糊泥，工作循环由原本的四阶段升级为**“两进两退”的八阶段全自动循环**：

```text
⓪ 建图扫描 → ① 吸尘下降 → ② 独立干吸(→) → ③ 升起返回(←) → ④ 清洗下降 → ⑤ 湿洗作业(→) → ⑥ 抽湿脱水(←) → ⑦ 自动卷收
```

| 阶段 | 动作名称 | 详细说明 |
| :--- | :--- | :--- |
| **⓪** | **视觉扫描** | 龙门架停靠起点，相机全局拍照生成污渍热力图坐标。 |
| **①** | **吸尘准备** | **第 1 排吸尘阵列独立下降**至距毯面 30mm。喷水与刷洗阵列保持高位收起状态。 |
| **②** | **独立干吸** | 龙门架正向（→）匀速运行。**纯干作业**，鸭嘴吸头按坐标精准吸除浮尘与干毛发。 |
| **③** | **空载复位** | 吸尘阵列抬起脱离毯面，龙门架高速反向（←）快速回到起点，为湿洗阶段准备。 |
| **④** | **清洗准备** | **第 2、3 排阵列联动下降**。此时吸尘阵列与抽湿阵列均保持高位安全区。 |
| **⑤** | **湿洗作业** | 龙门架再次正向（→）匀速运行。执行矩阵式局部喷水与模组化区域刷洗。 |
| **⑥** | **复合脱水** | 到达终点后，洗涤阵列抬起，**第 4 排抽湿阵列独立下降**。龙门架反向（←）慢速运行。真空强排拔水，高压风刀 45° 迎风剥离残余水膜。 |
| **⑦** | **自动卷收** | 龙门架回到起点。底部气动滚筒顶起，龙门架机械爪夹住毛毯末端推至卸料端，动力卷筒完成自动化收卷卸料。 |

### 五、渲染与输出文件

#### 5.1 技术方案
- **渲染引擎**：Three.js r165，PBR 材质（MeshStandardMaterial）体系。
- **后处理**：UnrealBloomPass 辉光 + FXAA 抗锯齿，支持实时阴影。
- **离线打包**：所有 Three.js 模块以 Blob URL 方式内嵌，单文件运行无需联网环境。
- **3D 交互**：OrbitControls 支持 360° 拖拽旋转、缩放、平移。

### 六、开发环境与编译脚本（仅供二次修改使用）

**文件目录结构：**
```text
B:\CAD\
├── js\                           Three.js r165 核心组件库（打包用）
├── carpet_washer_3d_v4.html      早期版本参考源码
├── carpet_washer_3d_v6_src.html  未压缩的 V3.1 开发源码（修改干湿分离逻辑从此切入）
├── build_v6_src.js               添加爆炸图解算矩阵的 Node 脚本
├── build_v6.js                   将源码及所有 JS 依赖打包内嵌的编译脚本
└── generate_dxf.js               基于当前尺寸常量重绘 DXF 几何的脚本
```
**二次开发流程：**
1. 采用 VS Code 编辑 `carpet_washer_3d_v6_src.html`，调整材质或更新 Z轴 A/B/C 的机械臂运动学逻辑。
2. 终端执行 `node build_v6.js` 生成最终免部署的单体 HTML。
3. 若修改了第二章节的物理尺寸，执行 `node generate_dxf.js` 重新输出更新后的 CAD 工程图纸。
