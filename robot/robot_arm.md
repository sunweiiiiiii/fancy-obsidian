# 机械臂设计全拆解

> 控制系统：Ubuntu 20.04
> 最后更新：2026-04-19

---

## 总体分层架构

```
[ 机械结构层 ]  →  [ 驱动执行层 ]  →  [ 感知层 ]  →  [ 控制计算层 ]  →  [ 软件程序层 ]
  关节 / 连杆        电机 / 驱动器       传感器           主控电脑            ROS2 / 算法
```

---

## 第一层：机械结构硬件

### 1.1 关节与连杆
| 组件 | 说明 | 常见材料 |
|---|---|---|
| **底座** | 固定整臂，承载全部重量 | 铝合金 / 钢 |
| **大臂（Link1）** | 第一段连杆，连接底座与小臂 | 铝型材 / 碳纤维 |
| **小臂（Link2）** | 第二段连杆 | 铝型材 / 碳纤维 |
| **腕部（Wrist）** | 末端姿态调整，通常 1~3 个自由度 | 铝合金 |
| **末端执行器** | 夹爪 / 吸盘 / 工具头，直接操作对象 | 依任务选型 |

### 1.2 自由度建议
| 场景 | 推荐自由度（DOF） |
|---|---|
| 教学 / 入门 | 4 DOF |
| 桌面通用操作 | 6 DOF（工业标准） |
| 灵活操作 / 冗余 | 7 DOF |

### 1.3 加工与采购
- **自制**：3D 打印（PLA / PETG）+ 铝型材，成本低，适合原型
- **采购**：乐高 Technic、大疆 RoboMaster、Trossen Robotics、亚博智能等套件
- **工业级**：UR 系列（Universal Robots）、KUKA、ABB

## 第二层：驱动执行硬件

### 2.1 电机选型
| 电机类型 | 特点 | 适用场景 |
|---|---|---|
| **舵机（Servo）** | 自带减速、位置反馈，接线简单 | 入门、小型臂（≤5kg） |
| **步进电机** | 开环控制、精度固定 | 中型臂、低成本方案 |
| **无刷直流电机（BLDC）** | 高转速、需编码器 + 驱动板 | 高性能臂 |
| **空心杯电机** | 体积小、响应快 | 轻量末端执行器 |
| **谐波减速器 + BLDC** | 高精度、高刚性 | 工业 / 科研级 |

### 2.2 驱动器 / 控制板
| 硬件 | 说明 |
|---|---|
| **Dynamixel 舵机总线** | 串行总线舵机，自带位置/速度/力矩反馈，与 ROS 集成好 |
| **ODrive** | 开源无刷电机驱动板，支持编码器闭环 |
| **Arduino / STM32** | 低层关节控制 MCU，接收上位机指令 |
| **Raspberry Pi 4** | 轻量上位机，可运行 ROS2 |
| **电机驱动模块（L298N / TB6600）** | 步进 / 直流电机驱动 |

### 2.3 电源系统
| 组件 | 建议规格 |
|---|---|
| 开关电源 | 24V / 10A 起（依电机数量） |
| 锂电池组 | 移动场景使用，22.2V 6S |
| 急停开关 | 安全必备 |
| 保险丝 / 过流保护 | 防止短路损坏驱动器 |

## 第三层：感知硬件（传感器）

| 传感器 | 用途 | 典型型号 |
|---|---|---|
| **编码器（Encoder）** | 关节角度位置反馈 | 磁编码器 AS5048、光电编码器 |
| **力矩传感器** | 末端力感知，防碰撞 | ATI Mini45、Rokubi |
| **IMU** | 姿态估计 | MPU6050、BNO055 |
| **深度相机** | 3D 视觉感知 | Intel RealSense D435、Azure Kinect |
| **RGB 相机** | 目标识别 | 罗技 C920、工业 USB 相机 |
| **限位开关** | 关节行程保护 | 微动开关 |
| **接触传感器** | 末端夹爪抓取感知 | FSR 压敏电阻 |

| **Ethernet** | 工业机器人控制器通信 |

## 第五层：软件程序层（Ubuntu 20.04）

### 5.1 操作系统配置
```bash
# 安装实时内核（提升控制实时性）
sudo apt install linux-lowlatency

# 设置串口权限
sudo usermod -aG dialout $USER
```

### 5.2 ROS2 安装（推荐 Foxy，对应 Ubuntu 20.04）
```bash
# 添加 ROS2 源
sudo apt install curl gnupg2 lsb-release
curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key | sudo apt-key add -
sudo sh -c 'echo "deb http://packages.ros.org/ros2/ubuntu focal main" > /etc/apt/sources.list.d/ros2.list'

# 安装 ROS2 Foxy
sudo apt update && sudo apt install ros-foxy-desktop

# 安装 MoveIt2（运动规划）
sudo apt install ros-foxy-moveit
```

### 5.3 软件栈分层
```
┌─────────────────────────────────────────┐
│         上层应用 / 任务规划               │
│   Python 脚本 / Web UI / 语音控制         │
├─────────────────────────────────────────┤
│         运动规划层                        │
│   MoveIt2 — 逆运动学、碰撞检测、路径规划   │
├─────────────────────────────────────────┤
│         ROS2 中间件层                    │
│   Topic / Service / Action 通信          │
├─────────────────────────────────────────┤
│         硬件抽象层（HAL）                 │
│   ros2_control — 统一接口驱动各类电机      │
├─────────────────────────────────────────┤
│         底层驱动                          │
│   Dynamixel SDK / ODrive / Arduino       │
└─────────────────────────────────────────┘
```

### 5.4 核心软件包
| 软件包 | 用途 |
|---|---|
| **ROS2 Foxy** | 机器人中间件，节点通信 |
| **MoveIt2** | 运动规划、逆运动学（IK） |
| **ros2_control** | 硬件接口标准框架 |
| **Gazebo / Ignition** | 物理仿真，先仿真再上真机 |
| **RViz2** | 3D 可视化调试工具 |
| **URDF / XACRO** | 机械臂模型描述文件格式 |
| **robot_state_publisher** | 发布关节状态到 TF 树 |
| **Dynamixel SDK** | Dynamixel 舵机通信库 |
| **OpenCV** | 图像处理 |
| **PCL（点云库）** | 深度相机点云处理 |

### 5.5 开发语言
| 语言 | 用途 |
|---|---|
| **Python 3** | 快速原型、任务逻辑、ROS2 节点 |
| **C++17** | 实时控制节点、底层驱动、性能敏感模块 |
| **XML / YAML** | URDF 模型、ROS2 参数配置 |
| **CMake** | ROS2 C++ 包构建系统 |
| **Bash** | 启动脚本、环境配置 |

## 第六层：开发流程建议

```
1. 确定 DOF 与任务需求
        ↓
2. 设计 / 采购机械结构
        ↓
3. 选型电机 + 驱动器，搭建电气系统
        ↓
4. 编写 URDF 模型，在 Gazebo 中仿真验证
        ↓
5. 编写 ros2_control 硬件接口，联调真实电机
        ↓
6. 配置 MoveIt2，测试运动规划
        ↓
7. 接入传感器（相机 / 力矩），开发感知模块
        ↓
8. 开发上层任务逻辑（抓取 / 搬运 / 装配）
```
